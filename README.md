# ChannelHashBot

## Setup

```bash
git clone https://github.com/YouTwitFace/ChannelHashBot.git
cd ChannelHashBot
cp .env.example .env
npm start
```

## .env

- **BOT_TOKEN** - the bot token from [@BotFather](https://t.me/BotFather)
- **COMMENTS_API_KEY** - the api key from [@CommentsBot](https://t.me/CommentsBot)
